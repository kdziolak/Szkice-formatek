# Ubiquitous Language

| Termin | Znaczenie |
| --- | --- |
| Droga | logiczna jednostka sieci drogowej identyfikowana numerem i klasa |
| Odcinek drogi | odcinek roboczy lub ewidencyjny drogi, do ktorego odnosza sie obiekty i raporty |
| Obiekt infrastruktury | element powiazany z droga, np. obiekt mostowy, przepust, urzadzenie BRD |
| Element techniczny | punktowy lub liniowy skladnik infrastruktury, np. znak, bariera, oswietlenie |
| Draft | roboczy zestaw zmian oczekujacy na walidacje i publikacje |
| Publikacja | zatwierdzenie zmian i przeniesienie ich do stanu referencyjnego |
| Referencja | slownik, identyfikator lub obiekt zewnetrzny, z ktorym laczymy dane robocze |
| Walidacja | zestaw reguł sprawdzajacych dane atrybutowe, topologie i kompletność |
| Ksiażka drogi | raport eksploatacyjny prezentujacy stan i zdarzenia dla odcinka drogi |
| Mapa techniczno-eksploatacyjna | mapa tematyczna budowana z opublikowanych warstw i atrybutow |
